How to Use

Add these tracking elements to your emails:

Open Tracking (Pixel)

Add this image tag to your email HTML (create an email record first):

<img src="{{base_url}}/{{tenant_id}}/pixel/EMAIL_ID.gif" width="1" height="1" style="display:block" />

Click Tracking

Replace your links with tracking URLs:

{{base_url}}/{{tenant_id}}/click/EMAIL_ID?url=https%3A%2F%2Fexample.com%2Fyour-link

Create Email Record

Use this API endpoint to create email records:

POST {{base_url}}/{{tenant_id}}/emails Content-Type: application/json { "subject": "Your email subject", "recipient": "user@example.com" }
